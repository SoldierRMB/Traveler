## 介绍
一个旅游景点订票系统

Spring Boot后端系统

## 快速开始
1. 克隆本项目
2. 执行 `divisions.sql` 和 `traveler.sql` 去创建数据库和表
3. 在 `application.yml` 中配置数据库和Redis连接
4. 使用Gradle构建
5. 启动Redis服务
6. 运行TravelerApplication

## 快速开始前端
1. 查看 [Traveler-Vue](https://github.com/SoldierRMB/Traveler-Vue)
